# Code of Conduct

This project should be a practical, respectful place to report issues, propose changes, and learn from each other.

## Expected Behavior

- Be kind and specific when giving feedback.
- Assume good intent, especially when someone is new to the project or the domain.
- Keep technical disagreement focused on the code, documentation, design, or user impact.
- Respect privacy. Do not post someone else's personal information, logs, tokens, or private files.
- Help keep issues and pull requests actionable by including context, reproduction steps, and constraints.

## Unacceptable Behavior

- Harassment, insults, threats, or sustained personal attacks.
- Discriminatory language or behavior.
- Sexualized comments or unwanted attention.
- Posting private information without permission.
- Deliberately derailing discussions or repeatedly ignoring maintainer guidance.

## Enforcement

Maintainers may edit, hide, lock, or remove comments and issues that violate this code of conduct. Maintainers may also block users from the project if needed to protect the community and the project.

If you need to report a conduct concern, open a private security advisory if appropriate, or contact the maintainer through the most direct private channel available on the GitHub profile.

## Scope

This code of conduct applies to project spaces such as issues, pull requests, discussions, documentation, and community support channels connected to this repository.
